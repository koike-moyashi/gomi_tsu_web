何をするの？

「分別さん」は津市のゴミ分別検索サイトで、どなたでも無料でご利用いただけます。

入力は「ひらがな」だけでも検索可。ゴミ分別パンフレットをパラパラする手間が省けます。

ゴミ分別検索に特化しているため、分別検索に関しては津市公式アプリよりも使いやすいかと思います（2024/5現在では元データは同じです）。

スマートフォンをお使いの方へ

デスクトップに はりつけておけば困ったときにさっと使え「ゴミパンフレットどこだっけ？」から解放されます。
