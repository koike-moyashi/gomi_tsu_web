このゴミ何ゴミ？を入力一つでお調べします

何をするの？

「分別さん」は津市のゴミ分別検索サイトで、どなたでも無料でご利用いただけます。

入力は「ひらがな」だけでも検索可。

ゴミ分別パンフレットをパラパラする手間が省けます。

スマートフォンをお使いの方へ

デスクトップに はりつけておけば困ったときにさっと使え「ゴミパンフレットどこだっけ？」から解放されます。

アプリのように使える設定はコチラ↓

ゴミ分別データについて（翻訳お手伝い募集中）

日本人でもややこしいゴミの分別。外国人の方はもっとお困りだと思います。

現在、英・中国・スペイン語に対応していますが、機械翻訳の為、まだまだ不便な状態です。
外国人の方々にも使っていただけるよう、みなさまのご協力をお願いします。
